OCP2PCIe

OCP (开放计算项目) 夹层网卡转标准 PCIe 槽转接卡设计方案.
OCP (Open Compute Project) Mezzanine card V2.0 to standard PCIe slot adapter design.

该项目可以将 OCP 夹层卡转接为标准 PCIe 卡方式供普通电脑使用.
This repo can convert OCP card to PCIe for PC use.

Link (链接)

https://github.com/KCORES/OCP2PCIe

For User (用户指南)

如果你想快速制作自己的OCP转接卡，请到发行页面下载已发布的 Gerber 文件包
并按用户指南准备相应的零件

If you want to build a OCP2PCIe card yourself quickly, Please visit Release to download Gerber zip package
